# Infrastructure

## Purpose
Contains all deployment and infrastructure configuration for production deployment.

## Responsibilities
- Containerization
- Orchestration
- Reverse proxy setup
- Environment configuration

## Key Components
- `docker/`: Dockerfiles for each service
- `docker-compose.yml`: Service orchestration
- `nginx.conf`: Reverse proxy configuration
- `scripts/`: Deployment scripts

## Tech Stack
- Docker
- Docker Compose
- Nginx
